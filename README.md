one_and_one
===========

1+1--one and one游戏<br>
<br>
这是款使用cocos2dx-js-3.0-rc制作的html5游戏。<br>
主要语言：JS
编译命令为oneandone目录下cocos compile -p web -m release<br>
<br>
游戏地址为：http://oneandone.luzexi.com 请使用手机查看。<br>
